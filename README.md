### From Extensions Website (Recommended)
1. Install the [GNOME Shell Integration browser extension](https://wiki.gnome.org/Projects/GnomeShellIntegration)
2. Visit [extensions.gnome.org/extension/9204/battery-power-mode-indicator/](https://extensions.gnome.org/extension/9204/battery-power-mode-indicator/)
3. Click the toggle to install and enable

### Manual Installation
```
git clone https://github.com/nirmaladvani/battery-power-mode-indicator.git 
```

```
cd battery-power-mode-indicator
mkdir -p ~/.local/share/gnome-shell/extensions/battery-power-mode-indicator@nirmaladvani.github.com
```
```
cp -r ./* ~/.local/share/gnome-shell/extensions/battery-power-mode-indicator@nirmaladvani.github.com/
```

X11: Restart GNOME Shell: Alt+F2, 'r', Enter

Wayland: Restart or logout and login back.
